# Real-Time Flight Data Project

This project extracts real-time flight data from the [AirLabs API](https://airlabs.co/).

## Introduction

This project utilizes the AirLabs Real-Time Flight Data API to obtain live flight information. The API provides a wide range of data parameters that can be retrieved, including:

### Parameters:

- `hex`: ICAO24 Hex address.
- `reg_number`: Aircraft Registration Number.
- `flag`: ISO 2 country code from Countries DB (Available in the Free plan).
- `lat`: Aircraft Geo-Latitude (Available in the Free plan).
- `lng`: Aircraft Geo-Longitude (Available in the Free plan).
- `alt`: Aircraft elevation (meters).
- `dir`: Aircraft head direction (Available in the Free plan).
- `speed`: Aircraft horizontal speed (km) (Available in the Free plan).
- `v_speed`: Aircraft vertical speed (km) (Available in the Free plan).
- `squawk`: Aircraft squawk signal code.
- `airline_icao`: Airline ICAO code.
- `airline_iata`: Airline IATA code.
- `aircraft_icao`: Aircraft ICAO type (Available in the Free plan).
- `flight_icao`: Flight ICAO code-number.
- `flight_iata`: Flight IATA code-number.
- `flight_number`: Flight number only.
- `dep_icao`: Departure Airport ICAO code.
- `dep_iata`: Departure Airport IATA code (Available in the Free plan).
- `arr_icao`: Arrival Airport ICAO code.
- `arr_iata`: Arrival Airport IATA code.
- `updated`: UNIX timestamp of the last aircraft signal.
- `status`: Current flight status - scheduled, en-route, landed.

## Usage

To use this project, simply make requests to the [AirLabs API](https://airlabs.co/) with the desired parameters to obtain real-time flight data.

## Acknowledgments

- Special thanks to [AirLabs](https://airlabs.co/) for providing the Real-Time Flight Data API.
